# AstraPetro Chemicals Static Demo

Dummy petrochemical corporate website for client presentation.

## Preview locally

```bash
python3 -m http.server 4173
```

Open:

```text
http://127.0.0.1:4173
```

## Deploy

This is a plain static website. Deploy the folder as-is.

- Vercel: import the folder/repo, framework preset `Other`, output directory `.`.
- Netlify: drag and drop the folder, or connect the repo with publish directory `.`.
- GitHub Pages: serve from the repository root.

## Demo content

All company names, product details, operating metrics, and enquiry flows are placeholder content for presentation only.
